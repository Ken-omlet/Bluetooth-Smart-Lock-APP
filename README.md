# 🔒Bluetooth-Smart-Lock-APP
## Overview
Bluetooth Smart Lock is an ESP32-based IoT security system that uses Bluetooth Low Energy (BLE) and RSSI-based proximity detection to automatically lock and unlock a door. The system authenticates trusted devices, monitors signal strength in real time, and provides a secure, hands-free access experience. An Android application interface has also been designed for future device management and activity monitoring.

## Feature
- Secure Bluetooth Low Energy (BLE) communication
- RSSI-based proximity monitoring
- Automatic lock and unlock functionality
- Activity log for lock/unlock events
- User authentication system
